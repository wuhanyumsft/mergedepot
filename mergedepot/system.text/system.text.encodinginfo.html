    EncodingInfo Class

    In this Article

    Provides basic information about an encoding.

    Remarks

    This class is primarily used by the Encoding class. The GetEncodings method returns an array of this type.

    This class is intended to provide minimal information about an encoding. To obtain additional information, the application should use the GetEncoding method to get an instance of the Encoding class, which contains more comprehensive information about the encoding it represents.

    • Name

      Gets the name registered with the Internet Assigned Numbers Authority (IANA) for the encoding.

      public string Name { get; }

      Property Value

      • String

        The IANA name for the encoding. For more information about the IANA, see www.iana.org.

      Remarks

      This property defines the name that is appropriate for passing to GetEncoding. This name is the same as the name represented by the WebName property. When its value is the name of a standard, the actual implementation of the encoding may not conform in full to that standard. Your applications should use DisplayName for a human-readable name.

      TheGetEncodings method gets a complete list of supported encodings, uniquely distinguished by code page. If your application retrieves encodings using the Name property, note that some duplicate encodings are retrieved. For more about handling these duplicates, see the description of GetEncodings.
