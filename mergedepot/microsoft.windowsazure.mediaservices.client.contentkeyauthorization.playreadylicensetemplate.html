    Properties summary

    AllowTestDevices
    AllowTestDevices

    Controls whether test devices can use the license or not. If true, the MinimumSecurityLevel property of the license is set to 150. If false (the default), the MinimumSecurityLevel property of the license is set to 2000.
    BeginDate
    BeginDate

    Configures the starting DateTime that the license is valid. Attempts to use the license before this date and time will result in an error on the client.
    ContentKey
    ContentKey

    Specifies the content key in the license. This is typically set to an instance of the ContentEncryptionKeyFromHeader object to allow the template to be applied to multiple content keys and have the content header tell the license server the exact key to embed in the license issued to the client.
    ExpirationDate
    ExpirationDate

    Configures the DateTime value when the the license expires. Attempts to use the license after this date and time will result in an error on the client.
    ExtensionData
    ExtensionData
    GracePeriod
    GracePeriod

    Configures the Grace Period setting of the PlayReady license. This setting affects how DateTime based restrictions are evaluated on certain devices in the situation that the devices secure clock becomes unset.
    LicenseType
    LicenseType

    Configures whether the license is persistent (saved in persistent storage on the client) or non-persistent (only held in memory while the player is using the license). Persistent licenses are typically used to allow offline playback of the content.
    PlayRight
    PlayRight

    Configures the PlayRight of the PlayReady license. This Right gives the client the ability to play back the content. The PlayRight also allows configuring restrictions specific to playback. This Right is required.
    RelativeBeginDate
    RelativeBeginDate

    Configures starting DateTime value when the license is valid. Attempts to use the license before this date and time will result in an error on the client. The DateTime value is calculated as DateTime.UtcNow + RelativeBeginDate when the license is issued
    RelativeExpirationDate
    RelativeExpirationDate

    Configures the DateTime value when the license expires. Attempts to use the license after this date and time will result in an error on the client. The DateTime value is calculated as DateTime.UtcNow + RelativeExpirationDate when the license is issued
