        A String object that represents the profile of the.NET Framework.

      Remarks

      A profile is a subset of the full .NET Framework. For example, the .NET Framework Client Profile is a subset of the .NET Framework 3.5 Service Pack 1, and the .NET Framework 4 Client Profile is a subset of the .NET Framework 4.5. For more information, see .NET Framework Client Profile.
