      Remarks

      If this option is true, the application can only be started from the user's Start menu. The default is false. This option is ignored if Install is false.

    • Install

      Gets or sets a value indicating whether the application is an installed application or an online-only application.

      public bool Install { get; set; }

      Property Value

      • Boolean

        true if the application is an installed application; otherwise, false.

      Remarks

      If this property is set to true, the application will be installed on the user's Start menu, and it can be removed by using Add or Remove Programs in Control Panel. The default is true. If this property is set to false, the application is intended for online use from a Web page and is available online only.

      Note

      For WPF Web Browser Applications, you must set the HostInBrowser property to true, and you must also set Install to false. If this condition is not met, you will receive MSBuild Error MSB3116.

    • MapFileExtensions

      Gets or sets a value indicating whether .deploy file extension mapping is used.

      public bool MapFileExtensions { get; set; }

      Property Value

      • Boolean

        true if .deploy file extension mapping is used; otherwise, false.

      Remarks

      If this property is set to true, every program file is published with a .deploy file extension. This setting is useful for Web server security to limit the number of file extensions that have to be unblocked to enable ClickOnce application deployment. The default is false.

    • MinimumRequiredVersion

      Gets or sets the minimum required version number for an application.

      public string MinimumRequiredVersion { get; set; }

      Property Value

      • String

        A string indicating the minimum required version number for an application.

      Remarks

      If the user's computer has a version of the application that is lower than the minimum required version specified by this property, the user will not have the option to skip the update. The default is to have no minimum required version. This input applies only when Install is set to true.

    • Product

      Gets or sets the name of the application.

      public string Product { get; set; }

      Property Value

      • String

        A string that indicates the name of the application.

      Remarks

      If no value is specified for this property, the name is inferred from the identity of the generated manifest. This name is used for the shortcut name on the Start menu and is part of the name that appears in Add or Remove Programs in Control Panel.

    • Publisher

      Gets or sets the publisher of the application.

      public string Publisher { get; set; }

      Property Value

      • String

        A string that indicates the publisher of the application.

      Remarks

      If no value is specified for this property, the name is inferred from the registered user or from the identity of the generated manifest. This name is used for the folder name on the Start menu and is part of the name that appears in Add or Remove Programs in Control Panel.

    • UpdateMode

      Gets or sets an object indicating whether updates should be checked in the foreground before the application is started, or in the background while the application is running.

      Property Value

      • UpdateMode

        An object that indicates whether updates should be checked before the application is started, or while the application is running.

      Remarks

      If this property is set to Foreground, updates are checked in the foreground. The default is Background. For more information, see UpdateMode.
