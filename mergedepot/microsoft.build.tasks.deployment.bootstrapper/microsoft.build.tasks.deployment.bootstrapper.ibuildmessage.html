    IBuildMessage Interface

    In this Article

    Represents messages that occur during the build operation.

    Syntax

    Declaration

    [System.Runtime.InteropServices.ComVisible(true)]
[System.Runtime.InteropServices.Guid("E3C981EA-99E6-4f48-8955-1AAFDFB5ACE4")]
[System.Runtime.InteropServices.InterfaceType(System.Runtime.InteropServices.ComInterfaceType.InterfaceIsDual)]
public interface IBuildMessage

    Properties summary

    HelpId

    Gets the MSBuild Help ID for the host IDE.
    HelpKeyword

    Gets the MSBuild F1 Help keyword for the host IDE.
    Message

    Gets a description of the details of the build message.
    Severity

    Gets the severity of the build message.

    Properties

    • HelpId

      Gets the MSBuild Help ID for the host IDE.

      public int HelpId { get; }

      Property Value

      • Int32

        An integer representing the MSBuild Help ID for the host IDE.

    • HelpKeyword

      Gets the MSBuild F1 Help keyword for the host IDE.

      public string HelpKeyword { get; }

      Property Value

      • String

        The MSBuild F1 Help keyword for the host IDE.

      Remarks

      The value of this property is the MSBuild F1 Help keyword for the host IDE, or null if no F1 Help is available.

    • Message

      Gets a description of the details of the build message.

      public string Message { get; }

      Property Value

      • String

        A text string that describes the details of the build message.

    • Severity

      Gets the severity of the build message.

      public Microsoft.Build.Tasks.Deployment.Bootstrapper.BuildMessageSeverity Severity { get; }

      Property Value

      • BuildMessageSeverity

        A object that represents the severity of the build message.
