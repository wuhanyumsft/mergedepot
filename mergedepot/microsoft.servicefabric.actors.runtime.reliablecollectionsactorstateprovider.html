      Parameters

      • stateManagerConfig
        ReliableStateManagerConfiguration
        ReliableStateManagerConfiguration

        A ReliableStateManagerConfiguration that describes IReliableStateManager configuration.

      • actorStateDictionaryCount
        Int32
        Int32

        Number of IReliableDictionary<TKey, TValue> across which actor states will be partitioned and stored.

      • reminderDictionaryCount
        Int32
        Int32

        Number of IReliableDictionary<TKey, TValue> across which reminders will be partitioned and stored.

      Remarks

      Values for actorStateDictionaryCount and reminderDictionaryCount can be specified only once when the Actor Service is created for first time. It cannot be changed after that and ReliableCollectionsActorStateProvider will ignore any values that are different from first time.
