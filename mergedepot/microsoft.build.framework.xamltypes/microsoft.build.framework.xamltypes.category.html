    • Subtype

      Gets or sets the subtype of this .

      public string Subtype { get; set; }

      Property Value

      • String

        To be added.

      Remarks

      The value of the subtype must be either Grid (default) or CommandLine.

      Subtypes help the UI display a category in an appropriate form. For example, category properties that are not command-line are typically displayed in the form of a property grid.
