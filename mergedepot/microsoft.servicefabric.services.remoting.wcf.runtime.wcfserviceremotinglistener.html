    WcfServiceRemotingListener WcfServiceRemotingListener Class

    In this Article

      public ServiceHost ServiceHost { get; }Public ReadOnly Property ServiceHost As ServiceHost

      Property Value

      • ServiceHost
        ServiceHost

        A ServiceHost used by this listener to host the WCF service implementation.

      Remarks

      The service host is created by the listener in it's constructor. Before this communication listener is opened by the runtime via OpenAsync(CancellationToken) method, the service host can be customized by accessing it via this property.
