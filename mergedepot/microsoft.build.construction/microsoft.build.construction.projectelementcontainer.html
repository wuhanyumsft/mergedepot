      Gets a recursive depth-first enumerator over all child elements.

      Property Value

      • IEnumerable<ProjectElement>

        Returns a recursive depth-first enumerator over all child elements.

      Remarks

      Recursion is possible because project element containers may contain children that are themselves project element containers.
