    AspNetCompiler Class

    In this Article

    Implements the AspNetCompiler task. Use the AspNetCompiler element in your project file to create and execute this task. For usage and parameter information, see AspNetCompiler Task.

    Syntax

    Declaration

      public bool AllowPartiallyTrustedCallers { get; set; }

      Property Value

      • Boolean

        true if the AllowPartiallyTrustedCallersAttribute should be applied to the strongly named assembly that Aspnet_compiler.exe generates; otherwise, false.

      Remarks

      This property must be combined with the KeyFile or KeyContainer properties.

    • Clean

      Gets or sets a Boolean value that indicates whether the precompiled application will be built clean.

      public bool Clean { get; set; }

      Property Value

      • Boolean

        true if the precompiled application will be built clean; otherwise, false.

      Remarks

      If this property is true, any previously compiled components will be recompiled. The default value is false.

    • Debug

      Property Value

    • FixedNames

      Gets or sets a Boolean value that indicates whether one assembly should be generated for each page in the application.

      public bool FixedNames { get; set; }

      Property Value

      • Boolean

        true if one assembly should be generated for each page in the application; otherwise, false.

      Remarks

      Each assembly is named with the virtual path of the original page unless the name would exceed the operating system limit for file names, in which case a hash is generated and used for the assembly name.

      You cannot use this property for in-place compilation; in-place compilation honors configuration settings for compilation batch mode.
