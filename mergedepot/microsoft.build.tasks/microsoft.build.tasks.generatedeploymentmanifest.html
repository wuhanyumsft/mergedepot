    GenerateDeploymentManifest Class

    In this Article

    Implements the GenerateDeploymentManifest task. Use the GenerateDeploymentManifest element in the project file to create and execute this task. For usage and parameter information, see GenerateDeploymentManifest Task.

    Declaration

    • CreateDesktopShortcut

      Gets or sets whether an icon is created on the desktop during ClickOnce application installation.

      public bool CreateDesktopShortcut { get; set; }

      Property Value

      • Boolean

        true if an icon is created on the desktop during installation; false otherwise.

    • DeploymentUrl

      Gets or sets the update location for the application.

      public string DeploymentUrl { get; set; }

      Property Value

      • String

        The update location for the application.

      Remarks

      If this property is not specified, no update location is defined for the application. However, if the UpdateEnabled property is true, an update location must be specified. The specified value should be a fully qualified URL or UNC path.

    • DisallowUrlActivation

      Gets or sets a Boolean value that indicates whether the application should be run automatically when it is opened through a URL.

      public bool DisallowUrlActivation { get; set; }

      Property Value

      • Boolean

        true if the application can only be started from the Start menu; otherwise, false.

      Remarks

      The default value is false. This property applies only when the Install property value is true.

    • ErrorReportUrl

      Gets or sets the URL of the Web page that is displayed in dialog boxes during ClickOnce installations.

      public string ErrorReportUrl { get; set; }

      Property Value

      • String

        A string that represents the URL of a Web page.

      Remarks

      Specifies a Web page to be displayed in dialog boxes that are displayed during ClickOnce installations.

    • Install

      Gets or sets a Boolean that indicates whether the application is an installed application or an online-only application.

      public bool Install { get; set; }

      Property Value

      • Boolean

        true if the application is to be installed on the local computer and can be removed by using the Add or Remove Programs dialog box; false if the application is for online use from a Web page.

      Remarks

      The default value is true.

    • MapFileExtensions

      Gets or sets a Boolean value that indicates whether the .deploy file name extension mapping is used.

      public bool MapFileExtensions { get; set; }

      Property Value

      • Boolean

        true if every program file is published with a .deploy file name extension; otherwise, false.

      Remarks

      Use this option for Web server security to limit the number of file name extensions that must be unblocked to enable ClickOnce application deployment. The default value is false.

    • MinimumRequiredVersion

      Gets or sets a Boolean value that indicates whether the user can skip the update.

      public string MinimumRequiredVersion { get; set; }

      Property Value

      • String

        true if the user can skip the update; otherwise, false.

      Remarks

      If a user has a version that is earlier than the minimum requirement, he cannot skip the update. This input only applies when the value of the Install property is true.

    • Product

      Gets or sets the name of the application.

      public string Product { get; set; }

      Property Value

      • String

        The name of the application.

      Remarks

      If this property is not specified, the name is inferred from the identity of the generated manifest. This name is used for the shortcut name on the Start menu and is part of the name that appears in the Add or Remove Programs dialog box.

    • Publisher

      Gets or sets the publisher of the application.

      public string Publisher { get; set; }

      Property Value

      • String

        The publisher of the application.

      Remarks

      If this property is not specified, the name is inferred from the registered user, or the identity of the generated manifest. This name is used for the folder name on the Start menu and is part of the name that appears in the Uninstall or change a program dialog box.

    • SuiteName

      Gets or sets the name of the folder on the Start menu where the application is located after ClickOnce deployment.

      public string SuiteName { get; set; }

      Property Value

      • String

        A string that represents the name of a Start menu folder.

    • SupportUrl

      Gets or sets the link that appears in the Uninstall or change a program dialog box for the application. The specified value should be a fully qualified URL or UNC path.

      public string SupportUrl { get; set; }

      Property Value

      • String

        The link that appears in the Uninstall or change a program dialog box for the application.

    • TrustUrlParameters

      Gets or sets a Boolean value that indicates whether URL query-string parameters should be made available to the application.

      public bool TrustUrlParameters { get; set; }

      Property Value

      • Boolean

        true if URL query-string parameters should be made available to the application; otherwise, false.

      Remarks

      The default value is false, which indicates that parameters will not be available to the application.

    • UpdateEnabled

      Gets or sets a Boolean value that indicates whether the application is enabled for updates.

      public bool UpdateEnabled { get; set; }

      Property Value

      • Boolean

        true if the application is enabled for updates; otherwise, false.

      Remarks

      The default value is false. This property only applies when the value of the Install property is true.

    • UpdateInterval

      Gets or sets the update interval for the application.

      public int UpdateInterval { get; set; }

      Property Value

      • Int32

        The update interval for the application.

      Remarks

      The default value is zero. This property only applies when the values of the Install and UpdateEnabled properties are both true.

    • UpdateMode

      Gets or sets a Boolean value that indicates whether updates should be checked. Updates can be checked before the application is started (in the foreground), or when the application is running (in the background).

      public string UpdateMode { get; set; }

      Property Value

      • String

        true if updates should be checked; otherwise, false.

      Remarks

      This property can have the following values:

      • Foreground (before the application is started)

      • Background (when the application is running)

        The default value is Background. This property only applies when the values of the Install and UpdateEnabled properties are both true.

    • UpdateUnit

      Gets or sets the time unit for the UpdateInterval property.

      public string UpdateUnit { get; set; }

      Property Value

      • String

        The time unit for the UpdateInterval property.

      Remarks

      This parameter can have the following values:

      • Hours

      • Days

      • Weeks

        This property only applies when the values of the Install and UpdateEnabled properties are both true.
