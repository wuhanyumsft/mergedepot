    CustomBehaviors
    CustomBehaviors

    This collection is initially populated by instantiation or by copying from the instantiating parent object (inheritance). In this model, the collections are independent but the members are shared references. Members of this collection alter or customize various behaviors of Azure Batch Service client objects. These behaviors are generally inherited by any child class instances.
    Modifications are applied in the order of the collection. The last write wins.
