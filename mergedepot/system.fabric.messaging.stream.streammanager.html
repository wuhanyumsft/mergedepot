    StreamManager StreamManager Class

    In this Article

    Edit

    The Stream Manager class is the primary entry point to Reliable messaging. The stream manager(singleton) is registered as a State Provider in State Manager and manages states(streams, sessions) using the the power of WinFab to provide the reliability and consistency guarantees for reliable messaging.
