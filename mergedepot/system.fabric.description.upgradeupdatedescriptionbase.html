    UpgradeUpdateDescriptionBase UpgradeUpdateDescriptionBase Class

    In this Article

    Edit

    Represents the base class for ApplicationUpgradeUpdateDescription and FabricUpgradeUpdateDescription. Used to modify the upgrade parameters describing the behavior of application or cluster upgrades. See UpdateApplicationUpgradeAsync(ApplicationUpgradeUpdateDescription) and UpdateFabricUpgradeAsync(FabricUpgradeUpdateDescription).
