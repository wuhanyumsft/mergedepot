    Task Class

    In this Article

    When overridden in a derived form, provides functionality for tasks.

    Remarks

    This abstract class provides default implementations for the methods and properties of the ITask interface.

    This class can only be instantiated in a derived form.

      Parameters

      • taskResources
        ResourceManager

        The task resources.

      • helpKeywordPrefix
        String

        The prefix to append to string resources to create Help keywords.

      Remarks

      This constructor allows derived task classes to register their resources, as well as provide a prefix for composing help keywords from string resource names. If is an empty string, string resource names will be used verbatim as help keywords.

      public Microsoft.Build.Framework.ITaskHost HostObject { get; set; }

      Property Value

      • ITaskHost

        The host object associated with the task. This value can be null.

      Remarks

      If the host IDE has associated a host object with this particular task, then the build engine sets this property.

      The host object is provided by HostServices. Visual Studio determines the host object via a system registry key. For more information, see <xref:Microsoft.VisualStudio.Shell.Interop.IVsMSBuildHostObject>.
