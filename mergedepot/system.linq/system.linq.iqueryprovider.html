    IQueryProvider Interface

    In this Article

    Defines methods to create and execute queries that are described by an object.

    Syntax

    Declaration

    public interface IQueryProvider

    Remarks

    The IQueryProvider interface is intended for implementation by query providers.

    For more information about how to create your own LINQ provider, see LINQ: Building an IQueryable Provider on MSDN Blogs.

      Note

      The Expression property of the returned IQueryable object is equal to .

      The CreateQuery method is used to create new IQueryable objects, given an expression tree. The query that is represented by the returned object is associated with a specific LINQ provider.

      Several of the standard query operator methods defined in Queryable, such as OfType and Cast, call this method. They pass it a MethodCallExpression that represents a LINQ query.

      Remarks

      The Execute method executes queries that return a single value (instead of an enumerable sequence of values). Expression trees that represent queries that return enumerable results are executed when their associated IQueryable object is enumerated.
